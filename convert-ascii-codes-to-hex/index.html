步骤及原理

  1. seq = list(asciiSeq)

    将原来的序列变成列表

  2. seq[0::2], seq[1::2]

    获取分别从0、1开始的间隔为 2 的序列

  3. zip(seq[0::2], seq[1::2])

    按位对齐这两个序列，生成一对一对的 tuple

  4. ''.join(l)

    连接

  5. chr(int(''.join(l), 16))

    • 以 base=16 转化上一步连接得到的字符串，生成 int

    • 转换成 char

  6. ''.join([chr(int(''.join(l), 16)) for l in zip(seq[0::2], seq[1::2])])

    把转换成的 char 连接起来就得到我们的字符串了

思考

其实本质是每两个一组，然后把其转换成对应的字符。
