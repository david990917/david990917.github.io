shell

shell是外壳的意思：保护着内核，同时作为和外界通信的桥梁。

shell是命令解释器，也是一种高级编程语言。

  • 命令解释器：解释和执行用户输入的命令
  • 高级编程语言：编写处可运行的脚本，提供函数和控制结构等语言编程能力

shell按照是否使用图形界面作为接口来分类：

  • 图形界面：Windows Explorer
  • 字符型：cmd、bash
shell 的特点：
  • 命中历史
    • history调用一使用过的命令
    • 重新打开的时候也会保存，保存在~/.bash_history文件 默认500条
  • 自动补全
    • 两次Tab罗列所有匹配的项目
  • 后台处理
    • 允许命令同时执行
  • 通配符功能
    • 快速找到文件
通配符：
  • 通配符？匹配任意一个字符
  • 通配符* 匹配任意多个字符
  • 字符范围符[]
    • 匹配任意一个中括号里面的字符
    • [12345] [1-5]
    • [a-zA-Z] 匹配所有的字母
配置shell：
  • shell的初始化
    • /etc/profile：为系统的每一个用户设置环境变量信息
    • ~/.bashrc：专用于当前用户的 Bash Shell信息（不需要root管理员权限）
    • /etc/environment：系统的环境变量
  • 添加环境变量
    • echo $PATH
编辑命令行：
  • 移动光标
    • CTRL+A：移动到开始位置
    • CTRL+E：移动到末尾位置
    • CTRL+F：光标向右移动一个位置
    • CTRL+B：光标向左移动一个位置
    • CTRL+l：清屏并从屏幕最上面开始新的一行
  • 编辑
    • CTRL+W：剪切光标前的一个单词
    • CTRL+Y：粘贴最近间切的字符串
    • CTRL+C：删除整行，并换到下一行
基本操作命令：

  • ls

    • .本级目录
    • ..上一级目录
    • -l 详细列出来
    • -R 递归显示下层子目录

  • cd

    • /开头是相对路径
    • ~ 表示shell用户的主文件夹
    • - 表示上次执行命令 cd 的目录

  • cat

    • 文本文件内容命令
    • -n 显示行号

  • grep

    • 匹配字符串命令

    • eg 在文件hello1中搜索包含字符“1”的所有文本行

  • sort 排序命令

    • -d 按照字典顺序排序
    • -f 忽略大小写
    • -r 逆序输出排序结果
文件和目录操作指令

  • 创建文件

    • touch

  • 复制文件

    • cp 源文件 目标路径

    • 备份文件

  • 删除文件

    • rm删除文件及整个目录

      • rm *.jpg
# 删除所有的jpg图像


  rm删除不可逆转

  rm -i 会询问是否删除

    • mv 移动命令

    • mkdir 创建目录命令
