Ubuntu16.04命令行安装Weblogic

安装之前

  • 需要安装 java

  • 在 Oracle 官网下载 Generic 安装包，下载完成上传到需要安装的服务器上

  • 创建weblogic用户

    赋予安装文件可执行的权限

开始安装

  • 切换到 weblogic 账户

  • 安装

    默认是 console 安装，因为我们也是命令行模式，所以正好。

    过程细节可以参见这个文章：linux安装weblogic10.3.6

我安装过程一切顺利，继续

启动

进入domain的安装目录

执行 startWebLogic.sh 文件

按照提示输入账户和密码

密码输入的时候是没有显示的。输入完成后回车确认。

如果用户名密码正确我们在最后会看到如下信息。

这个时候就说明我们的WebLogic以经启动了。

Weblogic

控制台输入：http://x.x.x.x:7001/console（若创建domain修改端口，输入你配置的端口）

输入输入创建域的用户名：weblogic，口令： weblogic123

打开这个页面的时候需要一定的时间。

上面就是装环境结束了，后面可能还需要一些高阶配置：
